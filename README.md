# 素材编辑器

基于PyQt6和OpenCV的图像合成工具，支持素材的拖拽、缩放、旋转等操作，具有完整的图层管理和项目保存功能。

## 功能特点

- **直观的图形界面**: 基于PyQt6的现代化界面设计
- **素材管理**: 支持从目录批量加载PNG素材文件
- **交互式编辑**: 鼠标拖拽、缩放、旋转素材
- **图层系统**: 完整的图层管理功能
- **属性编辑**: 实时调整素材的位置、缩放、旋转属性
- **项目管理**: 保存和加载项目文件
- **多种导出**: 导出合成图像和JSON标注文件
- **撤销重做**: 支持操作历史记录

## 安装依赖

```bash
pip install PyQt6 opencv-python numpy
```

## 使用方法

### 启动程序

```bash
# 方法1：直接运行主程序
python main.py

# 方法2：使用启动脚本（推荐）
python run.py
```

### 基本操作流程

1. **加载背景图像**
   - 菜单: 文件 -> 加载背景图像
   - 支持格式: PNG, JPG, JPEG, BMP, TIFF

2. **加载素材库**
   - 菜单: 文件 -> 加载素材目录
   - 自动扫描目录中的PNG文件
   - 支持mask文件（文件名包含"mask"）

3. **添加素材**
   - 在左侧素材面板选择素材
   - 在画布上点击位置放置素材
   - 可重复放置同一素材

4. **编辑素材**
   - 点击选择素材（显示选择框）
   - 拖拽移动位置
   - 鼠标滚轮缩放画布视图
   - 在属性面板调整具体数值

5. **图层管理**
   - 图层面板: 查看和管理所有图层
   - 添加/删除图层
   - 切换图层可见性
   - 调整图层顺序

6. **保存和导出**
   - 保存项目: 文件 -> 保存项目 (JSON格式)
   - 导出图像: 文件 -> 导出图像 (PNG/JPG)
   - 导出标注: 文件 -> 导出标注 (JSON格式)

### 快捷键

- `Ctrl+N`: 新建项目
- `Ctrl+O`: 打开项目
- `Ctrl+S`: 保存项目
- `Ctrl+Z`: 撤销
- `Ctrl+Y`: 重做
- `Delete`: 删除选中素材

## 素材格式要求

### 素材文件
- 格式: PNG（推荐，支持透明度）
- 命名: 任意名称.png
- 建议尺寸: 不超过2048x2048像素

### Mask文件（可选）
- 格式: PNG
- 命名: 原文件名 + "mask" + .png
- 例如: `apple.png` 对应 `apple_mask.png`
- 颜色: 白色为有效区域，黑色为无效区域

## 开发说明

本项目基于原始的fake_image.py代码开发，保留了核心的图像合成算法，并扩展为完整的GUI应用程序。

### 核心算法
- 支持NORMAL、MIXED_CLONE、COPY三种合成模式
- 图像变换（旋转、缩放）使用OpenCV实现
- 轮廓提取和分割点计算

### 架构设计
- MVC模式：分离数据模型、视图和控制逻辑
- 信号槽机制：组件间松耦合通信
- 图层系统：支持复杂的素材组织

## 故障排除

### 常见问题

1. **PyQt6导入失败**
   ```bash
   pip install PyQt6
   ```

2. **OpenCV导入失败**
   ```bash
   pip install opencv-python
   ```

3. **素材不显示**
   - 检查素材文件格式是否为PNG
   - 确认文件路径不包含中文字符
   - 查看控制台错误信息

4. **程序启动失败**
   - 使用`python run.py`启动以获得详细错误信息
   - 检查Python版本（建议3.8+）

### 性能优化建议

- 素材文件大小控制在1MB以内
- 避免同时加载过多大尺寸素材
- 定期清理不使用的图层和素材
