## Material Editor v2 使用说明

### 启动方式

- 确保已安装依赖：
  - `pip install -r requirements.txt`
- 在项目目录下运行：
  - `python main.py`

### 编译打包

本项目提供 PyInstaller 和 Nuitka 两套打包方式。PyInstaller 更轻量，Nuitka 更接近编译式打包但构建更慢。

- 安装运行依赖与构建依赖：
  - `python -m pip install -r requirements.txt`
  - `python -m pip install -r requirements-build.txt`
- 在目标系统本机执行构建：
  - PyInstaller：`python build.py`
  - Nuitka：`python build_nuitka.py`

构建结果：

- PyInstaller macOS Apple Silicon：`dist/MaterialEditor.app`
- PyInstaller Windows x86_64：`dist/MaterialEditor/MaterialEditor.exe`
- Nuitka macOS Apple Silicon：`dist-nuitka/MaterialEditor.app`
- Nuitka Windows x86_64：`dist-nuitka/main.dist/MaterialEditor.exe`

注意事项：

- macOS 版本需在 Apple 芯片 Mac 上使用 arm64 Python 构建。
- Windows 版本需在 64 位 Windows 上使用 64 位 Python 构建。
- PyInstaller 和 Nuitka 都不能可靠地跨系统交叉打包，请分别在 macOS 和 Windows 上运行同一套构建脚本。
- 当前打包脚本只内置核心功能（PySide6、numpy、OpenCV），不内置 LaMa、libcom、torch 等可选深度学习依赖。

### 界面布局

- **左侧：素材列表**
  - 显示通过“加载素材”导入的图片缩略图。
  - 双击某个素材，即可将其添加到中央画布。

- **中间：画布预览区**
  - 显示当前背景和所有素材叠加效果。
  - 支持：
    - 鼠标滚轮：以鼠标为中心放大/缩小。
    - 按住左键拖动：平移整个画布视图。
    - 点击素材：选中该素材，可在右侧调整属性。
    - 拖动素材：改变素材在画布中的位置。

- **右侧：已添加素材 + 属性 + 背景列表**
  - 上部“已添加素材”列表：
    - 列出当前画布上的所有素材。
    - 支持选中、删除、上移/下移、置顶/置底。
  - 中部“属性”：
    - 处理方式：直接粘贴 / 泊松融合 Normal / 泊松融合 Mix。
    - 旋转：滑条 + 数值 0–359°。
    - 缩放：滑条 + 数值 10–400%。
    - 颜色叠加：
      - 勾选“启用颜色叠加”生效。
      - “选择叠加颜色”按钮显示当前颜色。
      - “强叠加”：增强颜色影响力度。
      - “颜色透明度”：0–100%。
    - 掩码腐蚀/膨胀：负值缩小、正值扩展素材的掩码边界。
  - “背景与颜色”：
    - 背景取色器：点击后在画布上点选颜色，自动用于当前素材叠加。
    - 一键提取背景主色：从背景提取多种主色，在下方显示为颜色按钮，点击可快速应用到当前素材。
  - 底部“背景文件列表”：列出已加载的背景图，双击可切换。

### 工具栏功能

- **加载背景**：选择一个背景目录（或图片文件），递归扫描并加入右侧背景列表。
- **加载素材**：
  - 选择素材目录，支持自动识别 `A.png` 与 `A_mask.png`（掩码文件），并使用掩码作为透明通道。
  - 目录为空时可退回到手动多选文件。
- **上一张 / 下一张**：按列表顺序切换背景；切换背景时会清空画布上的素材。
- **清空素材**：移除画布和右侧列表中的所有素材。
- **一键导出**：
  - 导出当前背景 + 素材叠加的合成结果。
  - 泊松融合 Normal / Mix 模式在导出时使用 OpenCV `seamlessClone`，并附加强叠加色彩。
- **随机生成素材**：
  - 打开随机生成对话框，可配置生成数量、位置范围、旋转/缩放范围、混合模式和随机颜色叠加。
  - 自动从左侧素材中选取，支持简单的“避免重叠”策略。
- **高质量预览**（可勾选）：
  - 打开后，画布空闲时使用泊松融合在后台生成高质量预览。
  - 拖动素材或修改属性时会暂时关闭高质量图层，以保持交互流畅。
- **撤销 / 重做**：
  - 支持对添加/删除素材、移动、属性修改、图层顺序调整、背景切换等操作进行撤销/重做。

### 常用操作总结

- 加载背景 → 从右侧底部列表选中或用工具栏切换。
- 加载素材 → 左侧列表中双击素材添加到画布。
- 调整素材：
  - 在画布上拖动改变位置。
  - 在右侧属性中调节旋转、缩放、掩码腐蚀/膨胀。
  - 勾选“启用颜色叠加”，选择颜色并调整透明度、勾选“强叠加”增强效果。
  - 将处理方式设为“泊松融合 Normal/Mix”，并开启“高质量预览”以查看真实融合效果。
- 使用背景色：
  - 按“背景取色器”在画布中取色。
  - 或用“一键提取背景主色”生成多种主色按钮，点击快速应用。
- 批量随机布局素材：点击“随机生成素材”，配置范围与参数后点击“生成”。
